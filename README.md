# Testing APIs with Postman
For this challenge, two projects were created (Herokuapp and Reqres).

For the **Herokuapp project**, the file **"MyNewCollectionEnv.postman_environment.json"** represents the environment settings in Postman. The file **"My_New_Collection.postman_collection.json"** represents the collections in Postman. It also contains two .gif files with some evidence of the results and how the Newman file was executed.

For the **Reqres project**, the file **"ReqresEnvi.postman_environment.json"** contains the environment settings in Postman, and the file **"Reqres.postman_collection.json"** contains the collections in Postman.


Note: As I was not sent any files as requested in the PDF challenge, I took the liberty of using other APIs to complete it.
